## 🔑 WordPress Nedir?

Dünyanın en popüler **web sitesi kurma platformudur**.

```
İnternetin %43'ü WordPress kullanır!
PHP ile yazılmış + MySQL veritabanı kullanır
```

---

## 🔑 WordPress Kurulum (XAMPP)

```
1. XAMPP indir → apachefriends.org
2. Apache & MySQL başlat
3. WordPress indir → wordpress.org
4. C:/xampp/htdocs/wordpress/ kopyala
5. http://localhost/phpmyadmin → DB oluştur
6. http://localhost/wordpress → Kur!
```

## 🔑 XAMPP Nedir?

```
X → Cross Platform
A → Apache (Web sunucusu)
M → MySQL (Veritabanı)
P → PHP (Programlama dili)
P → Perl

Apache → Sayfaları sunar (Garson 🍽️)
MySQL  → Verileri saklar (Depo 📦)
PHP    → Kodu çalıştırır (Aşçı 👨‍🍳)
```

## 🔑 OpenCart Nedir?

Sadece **e-ticaret** için kullanılan platformdur.

```
WordPress → Her türlü site ✅
OpenCart  → Sadece e-ticaret ✅
```

### OpenCart Özellikleri:
```
✅ Çoklu dil desteği
✅ Çoklu para birimi
✅ Ürün yönetimi
✅ Sipariş takibi
✅ Ödeme entegrasyonu
✅ Kargo yönetimi
```
