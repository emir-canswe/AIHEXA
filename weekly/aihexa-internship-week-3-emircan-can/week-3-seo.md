## 1️⃣ SEO Nedir?

SEO, Search Engine Optimization yani Arama Motoru Optimizasyonu anlamına gelir. Bir web sitesinin Google gibi arama motorlarında üst sıralarda görünmesi için yapılan tüm çalışmaların genel adıdır.

**Neden önemlidir?**
- İnsanların %90'ı Google'ın ilk sayfasındaki sonuçlara tıklar
- İkinci sayfaya geçen kullanıcı sayısı çok azdır
- Üst sırada olmak daha fazla ziyaretçi demektir
- Daha fazla ziyaretçi daha fazla müşteri anlamına gelir

**SEO 3 ana başlıkta incelenir:**

```
1. On-Page SEO   → Sayfa içi optimizasyon
2. Technical SEO → Teknik optimizasyon
3. Off-Page SEO  → Sayfa dışı optimizasyon
```

---

## 2️⃣ On-Page SEO Nedir?

Sayfa içinde yapılan optimizasyon çalışmalarıdır. İçerik kalitesi, başlık etiketleri, URL yapısı ve görseller bu kategoriye girer.

**Anahtar Kelime:**

Kullanıcıların Google'da aradığı kelimelerdir. İçerikte doğal bir akışla kullanılmalıdır. Zorla anahtar kelime doldurmak Google tarafından cezalandırılır.

**Başlık (Title Tag):**

```
→ En önemli SEO faktörlerinden biridir
→ Maksimum 60 karakter olmalıdır
→ Anahtar kelime içermelidir
→ Her sayfa için farklı olmalıdır

✅ İyi örnek: "Java Full Stack Eğitimi | AIHEXA"
❌ Kötü örnek: "Sayfa 1"
```

**Meta Description:**

```
→ Arama sonuçlarında görünen açıklamadır
→ Maksimum 160 karakter olmalıdır
→ Kullanıcıyı tıklamaya teşvik etmelidir
→ Anahtar kelime içermelidir
```

**H1, H2, H3 Başlıkları:**

```
→ H1 sayfada sadece 1 tane olmalıdır!
→ H2 ve H3 alt başlıklar için kullanılır
→ Anahtar kelimeler başlıklarda yer almalıdır
→ Hiyerarşik sıralamaya uyulmalıdır
```

**İçerik Kalitesi:**

```
→ En az 300 kelime olmalıdır
→ Özgün içerik yazılmalıdır
→ Kopyala yapıştır yapılmamalıdır
→ Görseller ve tablolar kullanılmalıdır
→ İç ve dış linkler eklenmelidir
```

**URL Yapısı:**

```
✅ aihexa.com/java-egitimi
✅ aihexa.com/blog/react-nedir
❌ aihexa.com/page?id=123
❌ aihexa.com/sayfa1
```

**Alt Text (Resim Açıklaması):**

```
→ Her resme alt text eklenmelidir
→ Google resimleri okuyamaz, alt text sayesinde anlar
→ Anahtar kelime içerebilir
→ Açıklayıcı olmalıdır
```

## 3️⃣ Technical SEO Nedir?

Sitenin teknik altyapısının optimizasyonudur. Kullanıcı deneyimini ve arama motorlarının siteyi tarama kalitesini doğrudan etkiler.

**Site Hızı:**

```
→ 3 saniyeden hızlı yüklenmelidir
→ Yavaş site kullanıcıyı kaçırır
→ Google yavaş siteleri cezalandırır
→ Resimler sıkıştırılmalıdır
→ Önbellekleme (caching) kullanılmalıdır
```

**Mobil Uyumluluk:**

```
→ Responsive tasarım zorunludur
→ Google mobile-first indexing kullanır
→ Mobil uyumsuz site düşük sıralamaya düşer
→ Tüm cihazlarda test edilmelidir
```

**HTTPS:**

```
→ SSL sertifikası zorunludur
→ HTTP kullanan siteler güvensiz olarak işaretlenir
→ Google HTTPS'i tercih eder
→ Kullanıcı güvenini artırır
```

**Sitemap:**

```
→ Sitenin haritasıdır
→ Google'ın siteyi daha iyi taramasını sağlar
→ WordPress'te Yoast SEO otomatik oluşturur
→ Google Search Console'a gönderilmelidir
```

**Robots.txt:**

```
→ Google'a hangi sayfaları tarayacağını söyler
→ Gizli sayfaların indekslenmesini engeller
→ Admin sayfaları engellenmeli
→ Yanlış yapılandırma tüm siteyi gizleyebilir!
```

**Core Web Vitals:**

| Metrik | Açıklama | İyi Değer |
|--------|----------|-----------|
| LCP | Yükleme hızı | 2.5 saniyeden az |
| FID | Etkileşim hızı | 100ms'den az |
| CLS | Görsel stabilite | 0.1'den az |

## 4️⃣ Off-Page SEO Nedir?

Siteden bağımsız olarak yapılan çalışmalardır. Sitenin internet dünyasındaki otoritesini ve güvenilirliğini artırmayı amaçlar.

**Backlink:**

```
→ Başka sitelerden gelen linklerdir
→ Kaliteli backlink = Yüksek otorite
→ Spam backlink = Google cezası
→ Kalite miktardan önemlidir
```

**Sosyal Medya:**

```
→ İçerikleri sosyal medyada paylaşmak
→ Marka bilinirliğini artırmak
→ Siteye trafik çekmek
→ Dolaylı olarak SEO'ya katkı sağlar
```

**Google My Business:**

```
→ Yerel SEO için zorunludur
→ Google haritada görünmeyi sağlar
→ Adres, telefon, çalışma saatleri eklenir
→ Müşteri yorumları toplanır
```

## 5️⃣ WordPress'te SEO Nasıl Yapılır?

WordPress'te SEO için en yaygın kullanılan araç Yoast SEO plugin'idir.

**Yoast SEO ile her sayfa ve yazıda yapılması gerekenler:**

```
1. Focus keyword gir
2. Meta title yaz (max 60 karakter)
3. Meta description yaz (max 160 karakter)
4. URL düzenle
5. İçerikte anahtar kelime kullan
6. H1 başlık ekle
7. Görsellere alt text ekle
8. İç link ekle
9. Yeşil ışık al ✅
```

**Yoast SEO Renk Sistemi:**

```
🔴 Kırmızı → Düzeltilmesi şart!
🟡 Sarı    → İyileştirilebilir
🟢 Yeşil   → Mükemmel! ✅
```

## 7️⃣ Google Search Console Nedir?

Google'ın ücretsiz sunduğu, sitenin arama performansını takip etmeye yarayan araçtır.

**Google Search Console ile yapılabilecekler:**

```
→ Sitenin Google'da hangi kelimelerle göründüğünü gör
→ Kaç tıklama aldığını takip et
→ Hataları tespit et ve düzelt
→ Sitemap gönder
→ Manuel indeksleme isteği gönder
→ Mobil uyumluluk sorunlarını gör
```
